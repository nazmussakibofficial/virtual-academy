The most common difference between flexbox and grid is that flexbox is a one-dimensional layout and grid is a two-dimensional layout. It means that flexbox is best utilized when you have to display items in either a row or a column and grid is best utilized when you have to display items in both rows and columns.

Tailwind is more flexible and customizable than Bootstrap. But Tailwind requires deep knowledge about all the classes. On the other hand, boostrap already has pre-styled responsive components that are easier to build.

CSS Box Model is very important for understanding any CSS Framework. Any CSS Box containts four parts: content's height and width, padding, border and margin. Knowledge about these four parts are reqiured for customizing any css box.

A semantic tag is a html tag with a clear meaning. It helps search engine to determine the context of web pages. It has great accessibility for both users and developers.
